This is the csv simulation of the deutsch algorithm.

Input: Define the user input function in the 1_function_design.csv file. Since this is Deutsch algorithm, the number of qubits is fixed to 1.Enter the respective x (input) values in the second and the output (f(x)) values in the third column respectively.
Make sure to save the 1_function_design.csv file before executing the main code 2_deutsch.py.

2. Execute 2_deutsch.py for simulating the algorithm on your computer.

Enter your credentials(API Token, CRN) and use 2_deutsch_IBM.py for the IBM Implementation.

For both the above files, the number of shots is as the input.

make sure that you are in that particular directory where these files reside.

The output will show the type of function that was input with the shot counts.
