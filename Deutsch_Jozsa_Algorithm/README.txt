This is the csv simulation of the deutsch jozsa algorithm.

1. Input: Define the user input function in the 1_function_design.csv file. Since this is Deutsch Jozsa algorithm, the number of qubits is user specified in the csv file.
Enter the number of qubits in the first column, The x (input) values in the second and the respective output (f(x)) values in the third column respectively.
Make sure to save the 1_function_design.csv file before executing the main code (2_D-J_Algorithm.py).

2. Run the file 2_D-J_Algorithm.py for running the simulation on the local computer.

Enter your IBM Credentials and run the 2_D-J_Algorithm_IBM.py to run the same on IBM Quantum Computer.

For both the above files, the number of shots is as the input.

make sure that you are in that particular directory where these files reside.

The output will show the type of function that was input with the shot counts.(Constant, Balanced, NCNB).
